featured

BlitzKode

An open-source coding LLM I trained so it could run on a laptop without a GPU. Four-stage pipeline on Qwen2.5-1.5B-Instruct — SFT, reward SFT, DPO, then LoRA — which brought VRAM down from 24 GB to about 8. Exported to GGUF and served through FastAPI with SSE streaming. It's had 510+ downloads on HuggingFace.

    A document Q&A tool built around RAG. Each user gets their own FAISS index so retrieval stays isolated between accounts. Sub-2-second retrieval on 100k-token corpora, with JWT refresh-token rotation and Redis-backed revocation.

    A credit-card fraud detection API — 0.94 ROC-AUC on a RandomForest with 25 hand-built features. Includes Redis caching, MLflow tracking, drift detection, and a weekly retrain job. Runs in Docker Compose with Postgres, Prometheus, and Grafana.
